════════════════════════════════════════════════════════════════
ПЕРЕД ДЕПЛОЕМ (на локальной машине)
════════════════════════════════════════════════════════════════

[ ] 1. Git установлен? (проверка: `git --version`)

[ ] 2. GitHub аккаунт создан? (https://github.com)

[ ] 3. SSH ключи настроены?
    Команда: ssh -T git@github.com
    Должно быть: "Hi [username]! You've successfully authenticated"

════════════════════════════════════════════════════════════════
ИНИЦИАЛИЗАЦИЯ РЕПОЗИТОРИЯ (3 минуты)
════════════════════════════════════════════════════════════════

[ ] 4. Открыть PowerShell в корневой папке проекта:
    ```
    cd "c:\Users\kzdan\OneDrive\Desktop\Новая папка"
    ```

[ ] 5. Инициализировать Git:
    ```
    git init
    git add .
    git commit -m "Initial commit - v0.0.5 with payments"
    ```

[ ] 6. Создать репо на GitHub (https://github.com/new):
    - Name: tg-bot-v0.0.5
    - Public
    - Create repository

[ ] 7. Добавить remote и push код:
    ```
    git remote add origin https://github.com/YOUR_USERNAME/tg-bot-v0.0.5.git
    git branch -M main
    git push -u origin main
    ```
    (подставить свой username GitHub)

[ ] 8. Проверить: Код появился на GitHub? (https://github.com/YOUR_USERNAME/tg-bot-v0.0.5)

════════════════════════════════════════════════════════════════
РЕГИСТРАЦИЯ НА RENDER (2 минуты)
════════════════════════════════════════════════════════════════

[ ] 9. Зайти на https://render.com

[ ] 10. "Sign Up" → "Sign up with GitHub" → авторизация

════════════════════════════════════════════════════════════════
СОЗДАНИЕ WEB SERVICE (3 минуты)
════════════════════════════════════════════════════════════════

[ ] 11. "+ New" → "Web Service"

[ ] 12. "Connect a repository" → выбрать tg-bot-v0.0.5

[ ] 13. Заполнить форму:
        Name: tg-bot-v0.0.5
        Environment: Python
        Build Command: pip install -r bot_v0.0.2/requirements.txt
        Start Command: cd bot_v0.0.2 && python bot.py
        Plan: Free

[ ] 14. "Create Web Service"

[ ] 15. ЖДАТЬ BUILD (смотреть Logs) - примерно 3-5 минут

════════════════════════════════════════════════════════════════
ДОБАВЛЕНИЕ ПЕРЕМЕННЫХ ОКРУЖЕНИЯ (2 минуты)
════════════════════════════════════════════════════════════════

[ ] 16. На странице сервиса нажать "Environment"

[ ] 17. Добавить все переменные:
        BOT_TOKEN
        SUPABASE_URL
        SUPABASE_KEY
        CRYPTOBOT_TOKEN
        LOG_LEVEL
        (опционально) FREEKASSA_MERCHANT_ID, FREEKASSA_SECRET1, FREEKASSA_SECRET2

[ ] 18. "Save" → автоматический restart

════════════════════════════════════════════════════════════════
ПРОВЕРКА ЗАПУСКА (1 минута)
════════════════════════════════════════════════════════════════

[ ] 19. Смотреть Logs → ждать успешного запуска

[ ] 20. Увидеть URL сервиса вверху (https://tg-bot-....onrender.com)

════════════════════════════════════════════════════════════════
ОБНОВЛЕНИЕ WEBHOOK'ОВ (1 минута)
════════════════════════════════════════════════════════════════

        https://tg-bot-v0.0.5.onrender.com/webhook/freekassa
        
[ ] 22. FreeKassa → "СОХРАНИТЬ И ПРОВЕРИТЬ"
