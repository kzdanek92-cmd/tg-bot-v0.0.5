# ✅ Чек-лист тестирования

## Перед запуском

- [ ] Python 3.8+ установлен
- [ ] Установлены зависимости (`pip install -r requirements.txt`)
- [ ] Получен токен от @BotFather
- [ ] Токен вставлен в `bot.py` (строка 18)

---

## Базовые команды

### /start
- [ ] Бот отвечает приветствием
- [ ] Показывается список команд
- [ ] Форматирование корректное (жирный текст, эмодзи)

### /help
- [ ] Показывается та же информация, что и в /start

### /tasks
- [ ] Показываются все 5 заданий
- [ ] У каждого задания есть ID, название, описание, бюджет, категория
- [ ] Форматирование корректное

---

## Отклики на задания

### Успешный отклик
- [ ] `/respond 1` — отклик отправлен
- [ ] Показывается сгенерированный текст отклика
- [ ] Начисляется +50₽
- [ ] Показывается текущий баланс

### Повторный отклик
- [ ] `/respond 1` (второй раз) — ошибка "уже откликался"
- [ ] Баланс не изменился

### Несуществующее задание
- [ ] `/respond 999` — ошибка "задание не найдено"

### Неверный формат
- [ ] `/respond` (без ID) — ошибка "укажи ID"
- [ ] `/respond abc` — ошибка "ID должен быть числом"

---

## Баланс

### /balance
- [ ] Показывается текущий баланс
- [ ] Показывается количество откликов
- [ ] После откликов баланс увеличивается

---

## История откликов

### /my_responses (пустая история)
- [ ] Показывается сообщение "пока нет откликов"

### /my_responses (с откликами)
- [ ] Показываются все отправленные отклики
- [ ] У каждого отклика есть: название, дата, текст, заработок
- [ ] Показывается общий баланс

---

## Сохранение данных

### Перезапуск бота
- [ ] Отправить несколько откликов
- [ ] Остановить бота (Ctrl+C)
- [ ] Запустить снова
- [ ] Проверить `/balance` — данные сохранились
- [ ] Проверить `/my_responses` — история сохранилась

### Файл user_data.json
- [ ] Файл создается автоматически после первого отклика
- [ ] Файл содержит корректный JSON
- [ ] Данные пользователя сохраняются правильно

## Разные пользователи

### Тест с двумя аккаунтами
- [ ] Пользователь 1: отправить отклик, баланс = 50₽
- [ ] Пользователь 2: отправить отклик, баланс = 50₽
- [ ] Пользователь 1: проверить баланс — все еще 50₽
- [ ] Балансы не смешиваются

## AI-генерация

### Разные категории
- [ ] Отклик на "Копирайтинг" — соответствующий шаблон
- [ ] Отклик на "Креатив" — соответствующий шаблон
- [ ] Отклик на "Переводы" — соответствующий шаблон
- [ ] Отклик на "SMM" — соответствующий шаблон

### Случайность
- [ ] Создать нового пользователя

- [ ] Отправить отклик на задание 1
- [ ] Тексты откликов могут отличаться (случайный выбор из шаблонов)
